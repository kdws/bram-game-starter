# Legal and Privacy Notes

## IP cleanliness

This project should use original names, characters, art, music, dialogue, map layouts, and UI. It can be inspired by broad mechanics and moods, but should not copy specific expressive elements from Treasure MathStorm, Braid, Zelda, or any other game.

## AI-generated concept art

Use concept art as direction. For shippable assets, create cleaned, consistent production art with clear human art direction and documentation.

## Kids privacy

For the prototype:

- No accounts.
- No analytics SDKs.
- No ads.
- No chat.
- No geolocation.
- Local save only.

For a released product:

- Parental consent flow before collecting child personal information.
- Parent deletion/export controls.
- Minimal data collection.
- Clear privacy policy.
- No third-party advertising in the child experience.
