# Production Prompts

Use these for ideation and rough asset generation. Final production should still include human cleanup, consistency passes, and sprite-sheet packing.

## Global style prompt

Cozy hand-painted storybook fantasy game art, warm lantern light, mossy gardens, parchment UI, soft brush texture, layered parallax depth, gentle magical mood, readable child-friendly silhouettes, whimsical not scary, no gore, no graves, no horror, transparent background when making sprites.

## Bram side-view sprite prompt

A cute skeleton boy named Bram, child proportions, oversized expressive skull, small backpack, oversized dark hoodie, shorts, sneakers, warm golden glow accents, toy-like bones, cozy storybook game sprite, side view, clean readable silhouette, transparent background, animation-ready, no gore, no horror.

## Bram skateboard sprite prompt

A cute skeleton boy named Bram riding an enchanted skateboard, oversized skull, small backpack, cozy hoodie, child-friendly toy-like bones, warm glowing wheels, whimsical storybook fantasy game sprite, side view, transparent background, animation-ready, multiple poses for idle, push, ollie, kickflip, clean silhouette.

## Rattlewood background prompt

Cozy mossy forest village at golden hour, lanterns hanging from branches, small cottages, mushrooms, wooden bridges, soft mist, warm windows, magical floating clocktower in distance, painterly storybook 2D game background, parallax layers, no characters, 16:9.

## Top-down puzzle room prompt

Top-down cozy garden puzzle room, mossy stone tiles, lanterns, small pond, pressure plates, pushable crate, locked gate, coin puzzle area, friendly owl NPC space, readable grid layout, storybook fantasy game art, warm and gentle, not scary, 16:9.

## Parchment UI prompt

Hand-painted parchment game UI panel, rounded corners, warm dark brown interior, golden worn border, subtle leaf ornamentation, cozy fantasy, transparent background, no text.

## Important negative prompt

No blood, no gore, no horror, no graves, no realistic corpse, no adult skeleton proportions, no scary monster, no copied character, no brand logos, no copyrighted game UI, no existing game characters.
