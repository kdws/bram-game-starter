# BRAM: The Almost Boy — Playable Starter Prototype

This is a real starter repo for the BRAM concept: a cozy educational adventure with three gameplay modes.

## What is included

- Phaser + TypeScript + Vite project scaffold.
- Three prototype scenes:
  - **Rattle Run**: downhill skateboard math gates.
  - **Puzzle Platforming**: side-view math bridge puzzle.
  - **Top-Down Adventure**: exact-change garden puzzle.
- Local mastery tracking.
- Placeholder Bram drawing code.
- Concept art references.
- Art bible, asset manifest, production prompts, curriculum map, and production plan.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL Vite prints in the terminal.

## Controls

- Menu: click a mode.
- Rattle Run: left/up/right choose answer gates. M returns to menu.
- Platforming: arrows move, up/space jump. Click an answer button. M returns to menu.
- Top-down: WASD/arrows move and collect coins. M returns to menu.
